# Mortgage Tracker

Mortgage Tracker is a full-stack web application that allows users to submit mortgage applications and lets admins manage and review them.  

Frontend is built with **TypeScript + Next.js + Tailwind CSS + ShadCN UI**, and the backend uses **Node.js + Express + PostgreSQL + Knex.js**.  

The app is deployed on Heroku and Vercel.
